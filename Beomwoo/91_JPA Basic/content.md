### JPA 소개
- Java Persistance API
- Object-relational mapping
- 객체는 객체대로 설계, 관계형 데이터베이스는 관계형 데이터베이스대로 설계
- ORM 프레임워크가 중간에서 매핑
- JPA는 인터페이스의 모음(하이버네이트, EclipseLink, DataNucleus)
- SQL 중심적인 개발에서 객체 중심으로 개발이 가능해진다.
